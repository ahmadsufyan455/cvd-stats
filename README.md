# Covid Stats

### Features

1. Onboarding
2. Login by gmail with firebase auth
3. Show covid-19 statistic
4. Show covid-19 ratio chart
5. Localization (ID/EN)

### Library used

1. Glide
2. Retrofit
3. Viewpager2
4. Google auth
5. Dagger hilt
6. MPAndroidchart

Api from: https://api.covid19api.com/summary
